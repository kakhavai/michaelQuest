Install instructions.

1. Navigate to michaelQuest directory.
2. npm install
3. run with node server.js
4. server will be listening on your IP followed by port 4080
5. game will not begin until 2 players have connected. Can be initiated through opening multiple tabs.
6. A few easter eggs should probably be removed from game.js if this were to ever go to production. You can figure that out by seeing the on message responses.
7. Set the IP address to whatever the server is on in game.js
